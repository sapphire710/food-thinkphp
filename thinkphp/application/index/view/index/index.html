食全食美，食心食意，意境悠闲，意味不凡，三组食堂，让您宾至如归。
甜品
甜品，包括所有主菜后的食物，如布丁、煎饼、冰淇淋、奶酪、水果等等
主食
肉、禽类菜肴，最有代表性的是牛肉或牛排。常用烤、煎、铁扒等。
快餐
最早出现于德国，英语称为“quickmeal”或“fastfood”。西式快餐包括汉堡、披萨等
饮品
西餐的最后一道是上饮料，咖啡或茶。饮咖啡一般要加糖和淡奶油。茶一般要加香桃片（核桃仁和糯米等做成的糕）和糖。

餐厅简介 MORE+
2010年，来自深圳的贾先生来到山西创建了一家西餐茶食厅，当时营业面积只有70多平方米，主要经营西餐茶食小吃。改名为三组小食堂。“三组小食堂”以经营西餐为主，建筑和饮食风格收到广大顾客的欢迎。
目前餐厅营业面积1500平方米，员工120人。 一楼为现代派欧洲园林式酒吧风格；二楼为前苏联莫斯科“克里姆林宫”风格，设有高级单间；三楼为俄罗斯现代派风格，分南北两厅。现有四名特级西餐厨师亲自掌灶，这里的佳肴令您回味无穷、留连忘返。
